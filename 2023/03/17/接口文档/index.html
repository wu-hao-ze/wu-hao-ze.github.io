  • 调用方式： POST

content String 是 评论内容

  • 接口URL： /api/news
  • 调用方式： GET
  • 参数格式：无
  • 响应格式：
数据名称 数据类型 说明
status Number 200 获取新闻列表成功；500 获取新闻列表失败！
msg String 对 status 字段的详细说明
data Array 新闻列表数组
+id Number 新闻Id
+title String 新闻标题
+source String 新闻来源
+cmtcount Number 评论数量
+tags String 标签
+img String 图片地址
